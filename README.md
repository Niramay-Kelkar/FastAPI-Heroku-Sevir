# FastAPI-Heroku-Sevir
Deployment of SEVIR Nowcasting System for Insurance companies using Heroku

Fast api:
$ uvicorn apinew:app --reload


Streamlit:
$ streamlit run streamlit.py

It will run on http://localhost:8501/
